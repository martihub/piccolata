**********************************************
SALSA LipSync Suite - Actions for Behavior Designer
version 2.0.0
**********************************************

For lip sync use the Behavior Designer [Set Audio Clip] and [Play] Actions.

Emoter_Emote
	Activate/deactivate an EmoteR emotion.
Eyes_Blink
	Blink using the default timings or a one-time override.
Eyes_Look
	Set/clear a look target for tracking.
Eyes_SetAffinity
	Set/clear affinity and it's properties.
Eyes_SetRandom
	Set random head, eye, and blink states.
